# Unggul Mobile

Aplikasi mobile untuk CRUD Simple.

## Deskripsi

Project ini merupakan aplikasi mobile yang dibuat menggunakan Flutter.
Aplikasi ini mengimplementasikan Clean Architecture dan menggunakan BLoC untuk state management, get_it untuk dependency injection, dan dio untuk HTTP client.

## Instalasi

1. Pastikan Anda telah menginstal Flutter SDK. Untuk informasi lebih lanjut, silakan kunjungi [situs resmi Flutter](https://flutter.dev/docs/get-started/install).
2. Clone repository ini ke mesin lokal Anda.
   ```bash
   git clone https://github.com/example/unggul_mobile.git
   ```
3. Pindah ke direktori project.
   ```bash
   cd unggul_mobile
   ```
4. Install dependencies yang dibutuhkan.
   ```bash
   flutter pub get
   ```
5. Jalankan aplikasi.
   ```bash
   flutter run
   ``
   ```
